# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Waffo Go SDK, please **do not** open a public GitHub issue.

Instead, please report it by emailing:

**merchant.support@waffo.com**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours.

## Supported Versions

We provide security updates for the latest version of the SDK.
