# Contributing to Waffo Go SDK

Thank you for your interest in contributing!

## Reporting Bugs

Please open an issue with:
- A clear description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Go version and SDK version

## Suggesting Features

Open an issue describing the use case and proposed solution.

## Submitting Pull Requests

> **Note**: This repository is a read-only distribution mirror. The source code is developed in an internal monorepo. We do not accept code pull requests here. Please open an issue instead.

## Development Setup

```bash
go test ./...
```
